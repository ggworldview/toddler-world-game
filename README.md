# 🌍 寶寶認識世界 (Baby World Explorer)

這是一款專為兩歲以上幼兒打造的「跨平台高畫質單頁式認知學習網頁應用程式 (SPA)」。
透過 350+ 張最頂尖的真實物理相片與高品質插圖，加上極簡的拍立得視覺設計、動態場景切換與互動語音回饋，讓幼兒能在純淨、無干擾的頂級視覺環境下，快樂探索與建立認知地圖！

## ✨ 核心特色 (Core Features)

- 📸 **無與倫比的寫實圖庫**: 徹底屏除干擾認知與劣質的圖示 (Icons)，所有實體分類（動物、交通工具、家庭物品、蔬果）皆採用大量高畫質真實攝影相片。
- 📚 **循序漸進學程設計**: 獨創防重複演算法 (`pickSequentialItem`)，保證這多達 350+ 的詞彙能被平均吸收，拒絕隨機亂跳的學習盲區。
- 🎨 **動態情境沉浸場景**: 內建多重場景管理器機制，能即時根據寶寶切換至恐龍、昆蟲等不同分類時，自動變換對應的火山、森林等精美情境背景。
- ⚙️ **完全自治的擴張引擎**: 專案內建強大的 Python 非同步爬蟲模組，可以做到秒速擴增 100+ 新單字並全自動綁定到前端引擎中。
